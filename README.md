# 📊 Vendor Invoice Tracker — Excel VBA Automation

## 🔍 Project Overview
An Excel-based automation tool that streamlines vendor invoice 
management for finance and operations teams. Built to eliminate 
manual tracking, flag overdue payments, and auto-generate 
monthly PDF reports.

## ❓ Problem Statement
Finance teams in mid-size companies manually track 50–200 vendor 
invoices per month across spreadsheets — leading to missed due 
dates, delayed payments, and no real-time visibility on 
outstanding amounts.

## ✅ Solution
A single Excel tool that:
- Logs invoices via a structured entry form
- Automatically flags overdue payments with visual highlights
- Generates a dated PDF report in one click

## 🛠️ Tools & Skills Used
| Skill | Usage |
|-------|-------|
| Excel VBA | 3 macros — entry form, overdue highlighter, PDF export |
| Excel Macros | Automation of repetitive finance tasks |
| Process Automation | End-to-end: data entry → tracking → reporting |
| Dashboard Design | KPI cards, status breakdown, color coding |

## 📁 File Structure
- `Vendor_Invoice_Tracker.xlsm` — Main tool (macro-enabled)
- `Sample_Report.pdf` — Sample PDF output

## 🚀 How to Use
1. Open `.xlsm` file in Excel
2. Enable macros when prompted
3. Use **Alt + F8** to run:
   - `AddInvoice` — Add new vendor invoice
   - `HighlightOverdue` — Flag all overdue payments
   - `GeneratePDFReport` — Export dated PDF report

## 📌 Key Features
- Auto-generates Invoice IDs (INV-001, INV-002...)
- Color-coded status: 🟢 Paid | 🟡 Pending | 🔴 Overdue
- Dashboard with live KPIs — Total, Paid, Pending, Overdue amounts
- One-click PDF with today's date in filename
